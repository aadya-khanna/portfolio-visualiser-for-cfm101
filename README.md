# Portfolio Visualiser for CFM101 Group Project

This tool takes a group-submitted CSV file of portfolio holdings and visualises the resulting portfolio performance in comparison to benchmarks average such as the TSX and S&P 500 in real time.

The goal is stress out CFM101 students even after they've submitted the project to compare their portfolio during the duration of the competition. 

## HOW TO USE

- Go to (CFM101PortfolioVisualiser)[https://cfm101pv.streamlit.app/] 
- Input csv file in the formate of ticker, shares (final outputted csv)
- Track returns for our portfolio overtime!
